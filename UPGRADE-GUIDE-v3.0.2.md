# AIHub Works v3.0.2 升級說明

## 升級方式

1. 備份目前 GitHub Repository。
2. 解壓縮 `AIHub-works-v3.0.2.zip`。
3. 將解壓後的全部檔案覆蓋至 Repository 根目錄。
4. Commit 並 Push，等待 GitHub Pages 重新部署。
5. 使用瀏覽器無痕模式或強制重新整理檢查 `prompt-hub.html`。

## 本版主要變更

- Prompt Card 資訊層級、標籤、按鈕與間距重新設計。
- Modal 改為閱讀優先版面。
- 實戰案例改為獨立卡片。
- Modal 底部新增「關閉」按鈕。
- 手機版 Card 與 Modal 排版同步優化。

## 相容性

- 未修改提示詞與案例資料格式。
- 保留搜尋、收藏、最近瀏覽、熱門、分享、複製與鍵盤導覽功能。
- 不需要 npm、建置工具或資料庫。
